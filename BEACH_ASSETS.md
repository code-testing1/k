# Beach Scene Assets Download Guide

Download these professional assets to create a realistic beach finale scene:

## 🌌 **1. Night Sky Background**

**Download from Unsplash (Free):**

- Go to: https://unsplash.com/s/photos/night-sky-stars
- Search for: "night sky stars milky way"
- **Recommended images:**
  - https://unsplash.com/photos/milky-way-galaxy-wallpaper-E0AHdsENmDg
  - https://unsplash.com/photos/timelapse-photography-of-night-sky-uf2nnANWa8Q
  - https://unsplash.com/photos/silhouette-of-trees-near-aurora-borealis-aZjw7xI3QAA

**What to do:**

1. Click on image
2. Click "Download free" button (large size)
3. Save as: `night-sky.jpg`
4. Place in: `E:\Project\k\images\`

**Alternative source - Pexels:**

- https://www.pexels.com/search/night%20sky%20stars/
- Download any starry night sky image

## 🌙 **2. Realistic Moon (PNG with transparency)**

**Download from PNG Tree (Free with attribution):**

- Go to: https://pngtree.com/freepng/full-moon_5975307.html
- Or search: https://pngtree.com/so/moon-png
- Look for "realistic full moon PNG" with transparent background

**Download from Freepik (Free):**

- https://www.freepik.com/free-photos-vectors/moon-png
- Search for "realistic moon transparent"
- Choose PNG format with transparent background

**What to do:**

1. Download PNG with transparent background
2. Save as: `moon.png`
3. Place in: `E:\Project\k\images\`

**Alternative - Cleanpng:**

- https://www.cleanpng.com/
- Search: "full moon realistic"
- Download largest PNG size
